# Contributing to LangWeave

Help build LangWeave. Submit code, report bugs, share feedback.

## Contribution Methods

### Share the Project
Tell developers about LangWeave. Recommend it for internationalization needs. Growth benefits everyone.

### Submit Feedback
[Create an issue][02] with:
- Descriptive title
- Detailed description
- Specific use cases
- Improvement suggestions

### Report Bugs
[File a bug report][02] with:
- Clear title and reproduction steps
- Environment details and error messages
- Minimal reproduction example

### Contribute Code

#### Setup Environment
```shell
git clone https://github.com/sebastienrousseau/langweave.git
cd langweave
cargo build
cargo test
```

#### Submit Changes
1. Fork the repository
2. Create feature branch: `git checkout -b feature-name`
3. Edit files in `src/` directory
4. Write tests for new functionality
5. Run test suite: `cargo test`
6. Submit pull request with clear description

#### Code Requirements
- Follow Rust conventions
- Document public APIs
- Write tests for new features
- Maintain backward compatibility

## Development Process

### Pull Request Review
- Maintainer code review
- Multi-platform automated testing
- Documentation review for public changes
- Merge after approval and green tests

### Release Process
- Semantic versioning
- Changelog updates
- Pre-release testing
- Documentation updates
