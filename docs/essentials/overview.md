# Overview

LangWeave provides internationalization and localization for Rust applications.

## Capabilities

- **Language Detection** - Identify text language with confidence thresholds
- **Translation** - Translate between multiple language pairs
- **Content Management** - Manage localized content with caching
- **Performance** - Optimized algorithms for fast processing
- **Language Support** - Support for Latin, Cyrillic, Asian, Arabic scripts
- **Error Handling** - Comprehensive error types with recovery strategies

## Components

### Language Detection
`LanguageDetector` provides asynchronous language detection:
- Text analysis with configurable confidence thresholds
- Support for multiple writing systems
- Batch processing capabilities

### Translation Engine
Translation system features:
- Multi-language pair support
- Quality assessment scoring
- Contextual translation options

### Error Management
`I18nError` type provides:
- Detailed error classification
- Recovery strategy guidance
- Structured error information

## Use Cases

LangWeave works for:
- **Web Applications** - Multi-language user interfaces
- **CLI Tools** - Locale-adaptive command line tools
- **APIs** - International service endpoints
- **Content Management** - Localized content systems
- **Chat Applications** - Real-time language detection
