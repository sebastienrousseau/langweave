pub fn is_language_supported_optimized(lang: &str) -> bool

Optimized version of is_language_supported() with zero heap allocations.

Uses compile-time pattern matching for O(1) lookup without any allocations. Supports case-insensitive matching via ASCII case folding (no heap allocation).
